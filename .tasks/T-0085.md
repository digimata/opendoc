# T-0085 — Builder API

Ergonomic construction of Documents in Rust — what hermaeus extract modules will use.

## Scope

- Fluent builder for Document, blocks, inline content
- Avoid boilerplate for common patterns (paragraph with plain text, heading, simple table)
- Auto-generate stable IDs

## Done when

- Builder API works ergonomically for constructing Documents
- Tests demonstrate building a realistic multi-block document
- hermaeus extract modules can use this instead of manual struct construction
