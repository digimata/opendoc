# T-0086 — CLI — render and convert commands

Implement the `odoc` CLI with render and convert subcommands.

## Scope

```
odoc render filing.odoc -f md       # render to markdown
odoc render filing.odoc -f json     # canonical JSON
odoc convert input.html -o out.odoc # parse (future — domain-specific)
```

- `render` reads a `.odoc.json` file, outputs markdown or re-serialized JSON
- `convert` is a stub/future — actual HTML->odoc conversion is domain-specific (lives in hermaeus)

## Done when

- `odoc render <file> -f md` works end-to-end
- `odoc render <file> -f json` works end-to-end
- `odoc convert` prints a "not yet implemented" message
