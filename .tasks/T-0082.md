# T-0082 — Core types matching the spec

Implement the full type hierarchy from the [odoc spec](../docs/odoc.md) in Rust.

## Scope

- Document envelope (Document, Meta, Source, Version)
- Block types (Paragraph, Heading, List, Blockquote, Code, Table, Image, Divider, Embed)
- Inline nodes (Text, HardBreak)
- Marks (Strong, Em, Code, Link, Highlight, etc.)
- Annotations (Annotation, Target variants, Body)
- Provenance (ProvenanceRecord, kinds: Source, Assertion, Derived)
- Version metadata (VersionType, Change)

## Done when

- All spec types have corresponding Rust structs/enums in `odoc`
- Types derive `Debug, Clone, PartialEq`
- `cargo build` passes
