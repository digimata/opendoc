# T-0084 — Markdown renderer

Implement `odoc::render::markdown(&Document) -> String` — produces clean readable markdown from any Document.

## Scope

- Render all core block types to markdown (headings, paragraphs, lists, blockquotes, code blocks, tables, images, dividers)
- Render inline marks (bold, italic, code, links, etc.)
- Ignore annotations — pure content rendering
- Handle nested lists and blockquotes

## Done when

- `render::markdown` produces readable output from hand-built Documents
- Tests cover all block types
- Output is clean enough to drop into an LLM context window
